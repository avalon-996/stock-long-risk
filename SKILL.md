# Stock Long Risk - 股票多头持仓风险测评

分析股票多头产品持仓风险，计算极端情况下减仓所需时间和预期亏损。

## 功能特点

- 📈 **实时行情接入** - 使用腾讯财经接口获取 A 股实时行情数据（快速稳定）
- 📊 **整体风险评估** - 净值、盈亏、预警线/平仓线距离
- 📈 **持仓集中度分析** - 前5大重仓占比
- 🔗 **持仓相关性分析** - 计算股票间相关系数，识别同涨同跌风险
- 💧 **流动性分析** - 各持仓减仓所需天数
- ⚡ **极端情景模拟** - 市场下跌20%情景下的减仓过程
- 💰 **预期亏损计算** - 从触发平仓线到清仓的全程损益
- 📑 **Excel 报告导出** - 导出多维度风险分析数据到 Excel

## 安装依赖

```bash
pip install akshare pandas
```

## 风险指标

| 指标 | 说明 |
|------|------|
| 距离预警线 | 净值下跌到预警线所需的跌幅 |
| 距离平仓线 | 净值下跌到平仓线所需的跌幅 |
| 前5大集中度 | 前5大重仓占总市值比例 |
| 平均相关性 | 持仓股票间的平均相关系数（越低分散化越好）|
| 高相关性股票对 | 相关系数 > 0.7 的股票组合（同涨同跌风险）|
| 减仓天数 | 考虑流动性约束的减仓时间 |
| 极端情景亏损 | 市场下跌20%情景下的预期亏损 |

## 使用方法

### 命令行

```bash
# 使用示例数据（仅输出文本报告）
python3 scripts/stock_long_risk.py

# 使用自定义持仓文件
python3 scripts/stock_long_risk.py /path/to/holdings.json

# 导出 Excel 报告
python3 scripts/stock_long_risk.py /path/to/holdings.json /path/to/output.xlsx
```

### OpenClaw 中发送 Excel 到微信

在 OpenClaw 对话中使用：

```
请分析我的持仓风险并发送 Excel 报告到微信
```

AI 会执行：
1. 运行风险测评脚本生成 Excel
2. 使用 message 工具发送文件到微信

示例代码：
```python
# 生成 Excel 报告
result = generate_report('holdings.json', '/tmp/report.xlsx', 'o9cq805JNSBrHdVBZVY13ufbBRLQ@im.wechat')

# 发送文件到微信
message(action='send', to='o9cq805JNSBrHdVBZVY13ufbBRLQ@im.wechat', 
        message='📊 持仓风险测评报告', media='/tmp/report.xlsx')
```

### 持仓文件格式

```json
[
  {
    "code": "000001",
    "name": "平安银行",
    "shares": 10000,
    "cost_price": 10.50,
    "current_price": 11.02
  },
  {
    "code": "600519",
    "name": "贵州茅台",
    "shares": 100,
    "cost_price": 1650.00,
    "current_price": 1680.00
  }
]
```

> **注意**: `current_price` 字段仅在无法获取实时数据时作为 fallback 使用。正常情况下会自动获取最新行情。

### Excel 报告

导出文件包含以下工作表：

| 工作表 | 内容 |
|--------|------|
| 整体风险评估 | 总成本、总市值、盈亏、净值、预警线距离等 |
| 持仓明细 | 各股票的持仓数量、成本价、当前价、市值、盈亏 |
| 流动性分析 | 日均成交量、减仓天数、流动性评级 |
| 极端情景模拟 | 极端行情下的价格、卖出价、预计亏损 |
| 相关性矩阵 | 股票间的相关系数矩阵 |
| 高相关性股票对 | 相关系数 > 0.7 的股票组合 |
| **行业板块分布** ⭐v0.5.0 | 持仓股票的行业/板块分布统计 |
| **资金流向追踪** ⭐v0.5.0 | 基于涨跌幅的资金流向估算 |
| **综合风险雷达** ⭐v0.5.0 | 六维度风险评分雷达图 |

导出示例：
```bash
python3 scripts/stock_long_risk.py holdings.json /tmp/report.xlsx
# 输出: 📊 Excel 报告已导出: /tmp/stock_risk_report_20260329_130512.xlsx
```
